Как добавить обложки через загрузку файлов:
1. Перетащите изображения в зону ниже (или кликните)
2. Для каждого файла выберите товар из выпадающего списка
3. Нажмите «Загрузить все» — изображения сохранятся в БД и привяжутся к товарам

Поддерживаемые форматы: JPG, PNG, WEBP, GIF · Макс. размер: 6 МБ на файл
Совет: Называйте файлы по названию игры — например god-of-war.jpg

Как добавить обложки по ссылке:
1. Найдите изображение игры в интернете (Google Images, IGDB, PlayStation Store)
2. Скопируйте прямую ссылку на изображение (правой кнопкой → «Копировать адрес изображения»)
3. Вставьте ссылку в поле рядом с нужным товаром и нажмите «Применить»

Совет: Хорошие источники обложек: igdb.com, rawg.io, store.playstation.com
